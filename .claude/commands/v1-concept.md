# V1 컨셉

**모델**: opus (창작 판단)
**진행 표시**: `[1/6 컨셉확정]`

## 할 일

1. 소재에서 핵심 메시지 1문장 도출
2. 표현 방향 3안 제시, 각 안에 포함:
   - 제목/앵글
   - 톤 (정돈 / 가벼운 / 혼합)
   - 색상 팔레트 추천 (`src/constants.ts` PALETTES 기반 또는 신규 제안)
   - 음성 추천 (`references/gemini-voice-catalog.md` 참조)
   - 2~3문장 분위기 설명
3. **사용자 선택 대기** — 선택 전 다음 단계로 절대 넘어가지 않음

## 사전 참조
- `references/style-clean/index.md` (비주얼 톤 레퍼런스)
- `references/prompt-quality-standards.md` (스펙 품질 기준)
- spec-writing 스킬 (장면 태그 어휘)

## 관문
사용자가 번호를 선택하면 → 선택된 컨셉 확정 → 다음 단계

## 출력 포맷

```
[컨셉 1] {제목}
- 톤: {정돈/가벼운}
- 팔레트: {5색}
- 음성: {voice name}
- {분위기 설명}

[컨셉 2] ...
[컨셉 3] ...

어떤 방향이 좋으세요? (번호로 선택)
```

선택 후:
```
[1/6 컨셉확정] → 다음: 대본 작성
```
