> 🔄 V1 컨셉 시작

## 시리즈 분기

먼저 판단한다: **이 영상이 기존 시리즈의 연속편인가?**
- 같은 시리즈 폴더(`link-edu-series/` 등)에 이전 에피소드가 존재하면 → **시리즈 연속편**
- 아니면 → **신규 영상**

### 시리즈 연속편 → v1 스킵

시리즈 연속편은 팔레트·음성·톤·배경·레이아웃��� 전편과 동일하다.
대본이 확정되어 있으면 v1에서 결정할 것이 없다.

- **대본 확정** → v1 스킵, 바로 v3(비주얼 방향)으로
- **대본 미확정** → v1 스킵, 바로 v2(대본 작성)로

스킵 시 출력:
```
시리즈 연속편 — v1 스킵
계승: 팔레트 {이름} / 음성 {이름} / 톤 동일
대본 상태: {확정/미확정} → 다음: {v2 또는 v3}
```

> ✅ V1 컨셉확정 완료 (시리즈 계승)

### 신규 영상 → 3단계 순차 진행

#### 1단계: 교육 합의 (반드시 먼저)

사용자에게 다음을 **질문**한��. 한꺼번에 물어볼 것:

> **이 교육에 대해 알려주세요:**
> 1. **취지/목적** — 왜 이 교육을 만드나요?
> 2. **대상** — 누가 보나요?
> 3. **기대 효과** — 본 후 뭘 할 수 있게 되면 성공인가요?
> 4. **핵심 소재** — 어떤 내용을 다루��요?

사용자 답변을 받은 뒤 → 2단���로 진행.
**사용자가 답하기 전에 2단계로 절대 넘어가지 않음.**

#### 2단계: 컨셉 3안 제시

소재에서 핵심 메시지 1문장 도출 후, 표현 방향 3안을 제시한다.
각 안에 포함:
- 제목/앵글
- 톤 (정��� / 가벼운 / 혼합)
- 색상 팔레트 추천 (`src/constants.ts` PALETTES 기반 또는 신규 제안)
- 음성 추천 (`references/gemini-voice-catalog.md` 참조)
- 2~3문장 분위기 설명
- **비주얼 방향** (아래 카탈로그 판단 ��준 참��)

**사용자 선택 대기** — 선택 전 다음 단계로 절대 넘어가지 않음.

#### 3단계: 비주얼 방향 합의

사용자가 컨셉 번호를 선택하면, 해당 안의 비주얼 방향을 구체화하여 ���인받는다:

> **비주얼 방향 확인:**
> - 주력 카탈로그: {V1/V2/V5 조합}
> - 자주 쓸 장면 타입: {예: 키워드강조, 퍼널차트, 비교}
> - 배경 분위기: {예: 다크+파티클 적당히, 글로우 중앙}
> - 이 방향으로 대본 들어갈까요?

사용자 확인 후 → 컨셉 확정.

| 교육 성격 | 주력 카탈로그 | 이유 |
