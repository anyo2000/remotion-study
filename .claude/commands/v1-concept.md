# V1 컨셉

**모델**: opus (창작 판단)
**진행 표시**: `[1/6 컨셉확정]`

## 할 일 — 3단계 순차 진행

### 1단계: 교육 합의 (반드시 먼저)

사용자에게 다음을 **질문**한다. 한꺼번에 물어볼 것:

> **이 교육에 대해 알려주세요:**
> 1. **취지/목적** — 왜 이 교육을 만드나요? (예: 신입 FP 교육, 캠페인 홍보, 실적 개선)
> 2. **대상** — 누가 보나요? (예: 신입 FP, 경력 FP, 지점장, 고객)
> 3. **기대 효과** — 본 후 뭘 할 수 있게 되면 성공인가요? (예: 보장분석 프로세스 이해, 화법 체득)
> 4. **핵심 소재** — 어떤 내용을 다루나요? (키워드나 자료 위치)

사용자 답변을 받은 뒤 → 2단계로 진행.
**사용자가 답하기 전에 2단계로 절대 넘어가지 않음.**

### 2단계: 컨셉 3안 제시

소재에서 핵심 메시지 1문장 도출 후, 표현 방향 3안을 제시한다.
각 안에 포함:
- 제목/앵글
- 톤 (정돈 / 가벼운 / 혼합)
- 색상 팔레트 추천 (`src/constants.ts` PALETTES 기반 또는 신규 제안)
- 음성 추천 (`references/gemini-voice-catalog.md` 참조)
- 2~3문장 분위기 설명
- **비주얼 방향** (아래 참조)

**비주얼 방향**: 1단계 답변을 토대로, 카탈로그에서 이 영상에 어울리는 스타일을 조합한다.

판단 기준:
| 교육 성격 | 주력 카탈로그 | 이유 |
|----------|-------------|------|
| 숫자/데이터 설득 | V5(차트) + V1(숫자임팩트) | 객관적 근거 강조 |
| 프로세스/단계 설명 | V2(흐름도, 블록쌓기) + V5(퍼널, 타임라인) | 순서 시각화 |
| 공감/화법 교육 | V1(대화UI, 반전) + V2(이모지) | 감정 전달 |
| 비교/차별화 | V1(비교) + V2(스플릿스크린) + V5(비포애프터) | 대비 극대화 |
| 종합 교육 (긴 영상) | V1+V2+V5 혼합 | 장면 다양성 확보 |

**사용자 선택 대기** — 선택 전 다음 단계로 절대 넘어가지 않음.

### 3단계: 비주얼 방향 합의

사용자가 컨셉 번호를 선택하면, 해당 안의 비주얼 방향을 구체화하여 확인받는다:

> **비주얼 방향 확인:**
> - 주력 카탈로그: {V1/V2/V5 조합}
> - 자주 쓸 장면 타입: {예: 키워드강조, 퍼널차트, 비교}
> - 배경 분위기: {예: 다크+파티클 적당히, 글로우 중앙}
> - 이 방향으로 대본 들어갈까요?

사용자 확인 후 → 컨셉 확정.

## 사전 참조
- `references/style-clean/index.md` (비주얼 톤 레퍼런스)
- `references/prompt-quality-standards.md` (스펙 품질 기준)
- 카탈로그 컴포지션: VisualCatalog(V1), VisualCatalogV2, VisualCatalogV5

## 관문
1단계(교육 합의) → 2단계(컨셉 선택) → 3단계(비주얼 확인) — 3개 관문 모두 통과해야 완료.

1단계 질문 후:

```

2단계:
```

[컨셉 1] {제목}
