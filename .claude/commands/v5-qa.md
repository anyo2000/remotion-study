# V5 셀프 검수

**모델**: opus (품질 판단)
**진행 표시**: `[5/6 검수완료]`

## 할 일

### 5-1. 자동 검사
1. **폰트 크기 검증**: `node scripts/check-fontsize.js` 실행
   - 52px 미만 발견 시 → 사용자에게 보여주기 전에 수정
   - 동적 fontSize(변수, 계산식)도 최솟값 52 이상 확인
2. **팔레트 준수**: 모든 .tsx 파일에서 하드코딩된 색상 grep → constants.ts 팔레트 외 색상 없는지 확인
3. **중앙 정렬**: 레이아웃이 중앙축에서 벗어나지 않는지 확인

### 5-2. 스틸 캡처 검토
- 각 장면의 애니메이션 완료 시점에서 스틸 캡처:
  ```
  npx remotion still src/index.ts {CompositionName} out/still-{N}.png --frame={frame}
  ```
- 각 스틸에서 확인:
  - 텍스트 가독성
  - 레이아웃 균형
  - 색상 일관성
  - 정보가 음성보다 먼저 노출되지 않는지

### 5-3. 오디오 싱크 점검
- 텍스트 등장 타이밍 ≥ 나레이션 해당 부분 시작 시간
- 음성보다 먼저 정보 노출 금지

### 5-4. 사용자에게 제시
- 스틸 이미지 + 발견/수정 사항 보고
- 사용자 확인 대기
