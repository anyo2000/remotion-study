# 정돈 스타일 레퍼런스 (마케팅 트루먼쇼 채널)

## 출처
- 유튜브: 마케팅 트루먼쇼 (Claude Code 교육 영상 시리즈)
- 2026-03-29 저장

## 왜 이게 좋은가
- 정보 전달 장면에서 정돈감이 뛰어남
- 아이콘+텍스트 조합이 깔끔
- 요소 배치가 체계적

## 배울 점 (차장님 피드백 반영)

### 컨테이너 활용
- 프로세스/단계/비교 등 구조적 정보에 라운드 박스/원형 컨테이너 사용
- 단순 텍스트 장면에는 컨테이너 불필요 — 텍스트만으로 충분한 때가 있음
- 5.png: 3단계 프로세스 → 원형 아이콘 + 번호 뱃지 + 하단 텍스트
- 8.png: 비교 레이아웃 → 좌우 박스 + 화살표 + 프로그레스바

### 아이콘 vs 이모지
- 라인 아이콘: 프로세스, 기능 설명, 정돈된 장면에 어울림
- 이모지: 감정, 상황 묘사, 활기찬 분위기에 어울림
- 라인 아이콘이 못생기면 차라리 이모지가 나음 (상황별 판단)

### 텍스트 위계
- 제목: 크고 볼드 (흰색)
- 부제/설명: 중간 (흰색 또는 회색)
- 핵심 숫자/키워드: 강조색, 크게
- 보조 캡션: 작게, 회색
- 위계 차이가 명확할수록 정돈감 올라감

### 정렬
- 중앙축 정렬 철저 — 이건 절대 규칙

### 색상
- 이 레퍼런스는 검정+흰+초록 3색이지만, 차장님 영상은 3색 고정 아님
- 팔레트 사이트에서 계열을 잡되 원색/유치한 색 피하기
- 선명하면서 깨끗한 색 지향
