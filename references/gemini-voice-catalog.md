## 모델
- **Pro** (`gemini-2.5-pro-preview-tts`) — 품질 우선. 감정 태그/스타일 프롬프트 잘 반영
- **Flash** (`gemini-2.5-flash-preview-tts`) — 속도 우선. 품질 확연히 떨어짐. 쓸 이유 없음

## 샘플 파일 위치
- `public/audio/test-s5-{음성명}.wav` — "4월. 곧 찾아갑니다." 동일 문장 8개 음성 비교
- Flash 모델로 생성됨 (Pro 재생성 시 품질 더 좋음)

## 스타일 프롬프트 팁
Pro 모델은 텍스트 앞에 연출 지시를 붙이면 품질이 크게 올라감:
```
자신감 있고 따뜻한 나레이터. 보험 교육 티저 영상.
절제된 톤이지만 핵심 숫자에서 살짝 멈추고 강조.
---
(본문 텍스트)
```

## 감정/연출 태그 (Pro에서만 잘 작동)
- `[pause]` — 짧은 쉼
- `[speaking slowly]` — 천천히
- `[confident]` — 자신감
- `[warm, building anticipation]` — 따뜻 + 기대감
- `[whisper]`, `[excited]`, `[sigh]` 등

## 판단 기준 (2026-04-04 기준)
- **티저/광고**: Leda 또는 Zephyr
- **교육 내레이션**: Leda 또는 Kore
- **짧은 임팩트 멘트**: Zephyr
- **남성 음성 필요 시**: Charon (무게감) 또는 Fenrir (힘)
