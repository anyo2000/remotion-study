# LINK 컨설팅 교육 영상 에피소드 목록

> **총 ~21개**, 각 2~5분, 세로 9:16 (1080×1920)
> **원본**: 지점장 교육 대본 v2 (45 슬라이드) + 교안 PPTX
> **구조**: 단계별 [원리 1개 + 실전 화법 여러 개]

---

## 오리엔테이션 (2개)
1. **왜 옛날 영업이 안 되는가** — 의리가입 종말, 대면기피, SFP 현실 (슬라이드 3~5)
2. **LINK 4단계 전체 지도** — 오리엔테이션 (슬라이드 6~9)

## L단계 — 후킹 (5~6개)
3. **후킹의 원리** — 질문의 힘, 지하철 이야기, 3초 (슬라이드 15) ← 첫 번째 제작 완료
4. **경계심 풀기** — 보장분석 대신 이렇게, 단어 바꾸기 (슬라이드 10~14)
5. **후킹화법 1**: 임신출산 컨셉 (슬라이드 16)
6. **후킹화법 2**: 암 치료비 무한리필 (슬라이드 17)
7. **후킹화법 3**: 로봇수술비 (슬라이드 18)
8. **후킹에 반응 없을 때** (슬라이드 19)

## I단계 — 진단 (4~5개)
9. **진단의 원리** — 숫자가 대신 말한다, 분석지 역할 (슬라이드 20~22)
10. **소액암 분석지 실전** (슬라이드 23~25)
11. **암보장 분석지 실전** (슬라이드 26~27)
12. **실손 분석지 실전** + 5세대 (슬라이드 28~29)
13. **인증번호 자연스럽게 받기** — I→N 브릿지 (슬라이드 30~31)

## N단계 — 설계 (3~4개)
14. **설계의 원리** — 제안서 꺼내면 컨설팅이 판매로 (슬라이드 32~34)
15. **예산 제안** — 나이×만원 + 구찌/MCM + 뷔페 (슬라이드 35~37)
16. **우선순위 담보 선택 + 소액 플랜** (슬라이드 38~39)

## K단계 — 클로징 (3~4개)
17. **클로징의 원리** — 고객이 했던 말이 무기 (슬라이드 40)
18. **거절처리 1**: 건강비유 (슬라이드 41)
19. **거절처리 2+3**: 옷장비유 + 수도꼭지 (슬라이드 42)
20. **판매하지 말고 해결해드리세요** (슬라이드 43)

## 마무리 (1개)
21. **이번 주에 이것만 해보세요** (슬라이드 44~45)

## 에피소드 유형별 장면 패턴

| 에피소드 유형 | 주요 장면 구성 |
|-------------|--------------|
| **원리** (3,9,14,17) | 사례 → 왜 그런가 → 핵심 키워드 → 마무리 |
| **화법 실전** (5~8,18~19) | 상황 → 잘못된 예(X) → 올바른 화법(O) → 고객 반응 → 연습 팁 |
| **도구 실전** (10~12) | 도구 소개 → 단계별 사용 → 실전 대화 → 정리 |
| **마인드셋** (1,20) | 현실 제시 → 왜 바뀌어야 하는가 → 방향 |
| **오리엔테이션** (2) | 전체 구조 → 각 단계 한 줄 → 다음 영상 예고 |
