# LINK 컨설팅 티저 (숫자 충격형)

## 컨셉
- 스타일: 숫자 충격 → 공감 → 암시 → 브랜드
- 팔레트: coolBlue(S1-S3) → orange(S4-S5)
- 음성: Gemini Pro TTS, Leda
- BGM: Lyria 3 Clip, 120 BPM

## 대본

**S1 — 숫자 충격** (0s~9.1s)
요즘 10명 상담하면 몇 건 정도 체결하시나요? 2건? 1건? 그럼 나머지 고객은, 어디서 떨어진 걸까요?

**S2 — 벽** (9.1s~14.9s)
관심 없다는 고객, 인증번호를 안 주는 고객, 생각해볼게요로 끝나는 고객.

**S3 — 전환** (14.9s~19.7s)
우리가 매일 마주치는 이 벽을, 단계별로 부수는 방법이 있습니다.

**S4 — LINK** (19.7s~24.6s)
관계가 아니라 실력으로 계약하는 방법, 링크 컨설팅.

**S5 — 엔딩** (24.6s~29.5s)
4월. 곧 찾아갑니다.
